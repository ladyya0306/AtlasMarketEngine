# 发布前人工界面验收记录（2026-04-18）

这份记录只回答一个问题：

**用户当前能看到的界面和入口，是否已经符合发布口径。**

---

## 1. 验收范围

本轮界面验收覆盖两部分：

1. Scholar CLI 对外可见提示
2. Web 对外可见页面与启动入口

本轮验收刻意**不**覆盖：

1. 内部数据库字段迁移
2. 内部兼容层 `month/months` 的统一改名

本轮验收使用本地 Web 服务：

- `http://127.0.0.1:8018/web/index.html`

动态接口验收：

- `http://127.0.0.1:8018/config/schema`

1. `http://127.0.0.1:8018/web/index.html`
   - 返回 `200`
2. `http://127.0.0.1:8018/config/schema`
   - 返回 `200`

### 2.2 页面上已确认存在的关键控件和文案

在 Web 页面源码和实际返回内容中，已经确认能看到：

1. `固定供应盘`
2. `需求倍率`

4. `回合末人工干预面板`

从页面和接口默认配置看，当前启动顺序已经与发布口径一致：

1. 先选固定供应盘
2. 再设需求倍率
3. 再决定是否加入自动冲击

从 `/config/schema` 返回的 `startup_defaults.release_startup.supply_snapshots` 中，已经确认同时存在：

1. 梭子型固定供应盘可选
2. 金字塔型固定供应盘也可选
