# 公开发布政策（2026-04-18）

## 总原则

本仓采用 **只公开衍生证据（Derived Evidence Only）** 的发布政策。

目标是同时满足两件事：

1. 外部使用者能理解项目是什么、为什么、怎么跑。
2. 不直接公开内部完整原始研究资产。

## 公开内容

允许公开：

- 项目运行代码
- Web / CLI 入口
- 配置文件
- 测试脚本和 clone 自检脚本
- 汇总结果
- 通俗总结
- 说明文档
- 操作手册
- 演示材料
- 关键样例证据

## 不公开内容

不公开：

- 内部原始运行数据库
- 完整原始日志包
- 内部实验大包
- 私有环境变量
- 私有研究草稿和内部沟通台账

## 公开仓中看到的“结果”

如果公开仓里提到某组实验结果，请理解为：

- 已经从内部运行样本中提取出的结论
- 只保留支撑结论所需的摘要和样例
- 不代表公开仓附带了整包原始实验数据

## 关于“月份 / 回合”

本公开仓对外统一使用“回合（Round）”。

如果在代码、数据库字段或兼容性输出里仍看到 `month/months`，应理解为内部兼容字段名，不应直接理解为现实世界自然月。
