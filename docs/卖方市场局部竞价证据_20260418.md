# 卖方市场局部竞价证据（2026-04-18）

这份附录只回答一个问题：

**为什么 `seller_market` 看起来更偏卖方，但平均总成交价却没有高过自然基准？**

最短答案是：

1. 局部热点房源确实发生了更强竞价
2. 竞价机制没有失效
3. 但整条样本里又卖掉了更多低总价主流盘
4. 所以“局部抬价”被“成交结构下沉”在均价上盖过去了

## 2. 应该看数据库的哪几张表

如果要复查这条结论，最关键的是下面四张表：

1. `property_buyer_matches`
   - 看谁参与了同一套房的竞争
   - 重点字段：
     - `listing_price`
     - `buyer_bid`
     - `final_outcome`
     - `failure_reason`
     - `order_id`
     - `final_price`
2. `transactions`
   - 看最后真正成交的是哪套房
   - 重点字段：
     - `property_id`
     - `final_price`
     - `month`
     - `order_id`
3. `properties_static`
   - 看房子属于哪类区位和产品
   - 重点字段：
     - `zone`
     - `property_type`
     - `is_school_district`
     - `building_area`
4. `profiled_market_property_buckets`
   - 看这套房在供给画像里属于哪个桶位
   - 重点字段：
     - `bucket_id`
     - `metadata_json`

## 3. 先看总量证据：多人竞价确实更强

`seller_market` 和自然基准相比，关键差别如下：

| 指标 | `seller_market` | 自然基准 |
|---|---:|---:|
| `Outbid` 落败总数 | 89 | 41 |
| 同一套房同一轮有多人参与竞争的房源回合组合 | 149 | 104 |
| 来自这些“多人竞争房源”的最终成交套数 | 18 | 13 |
| 多人竞争成交房源的成交价 / 挂牌价均值 | 1.0209 | 0.9957 |

这说明：

1. `seller_market` 不是没有竞价
2. 相反，它的局部竞价比自然基准更强
3. 机制并没有失效

---

## 4. 再看结构证据：为什么均价没被整体抬高

`seller_market` 和自然基准的成交结构，对比如下：

| 成交层级 | `seller_market` 占比 | 自然基准占比 |
|---|---:|---:|
| 入门盘 | 32.3% | 14.7% |
| 改善盘 | 48.4% | 62.7% |
| 家庭大户型 | 18.3% | 20.0% |
| 高端盘 | 1.1% | 2.7% |

这说明：

1. `seller_market` 多卖掉了很多低总价主流盘
2. 改善盘和高端盘占比反而下降了
3. 所以“局部热点房被抬价”这件事，被“成交结构下沉”在均价上稀释掉了

## 6. 最后怎么解释这组现象

这组结果最准确的解释不是：

- “卖方市场失败了”

而是：

- “卖方市场主要强化了局部热点房源的竞价能力，但整条样本卖掉了更多低总价主流盘，所以整体平均总成交价没有同步抬高。”

因此，发布时解释 `seller_market`，不能只盯一个“平均总成交价”，更应该一起看：

1. `Outbid` 总数
2. 多人竞价成交房源数
3. 多人竞价成交房源的成交价 / 挂牌价
4. 成交结构有没有明显向低总价主流盘下沉
