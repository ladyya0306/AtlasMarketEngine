# 发布目录索引（公开仓）

这份索引只回答两个问题：

1. 公开仓现在有哪些发布材料
2. 这些材料应该按什么顺序看

> 说明：本公开仓采用“只公开衍生证据”的发布政策，不公开内部原始数据库、完整日志包和完整实验大包。
>
> 另一个边界也请注意：部分详细证据文档里仍会保留原始内部取证路径，这些路径是“来源追溯信息”，不是要求公开仓本地必须存在的目录。

## 第一次访问，先点这 3 个入口

1. [README.md](/D:/GitProj/AtlasMarketEngine/README.md)
   - 先快速理解项目是什么、为什么存在、能做什么
2. [public_smoke_test.py](/D:/GitProj/AtlasMarketEngine/scripts/public_smoke_test.py)
   - 先验证 clone 后最小可运行链条
3. [START_WEBSITE.md](/D:/GitProj/AtlasMarketEngine/START_WEBSITE.md)
   - 如果你想先看网页界面，从这里开始

如果只想看发布材料，再往下按“推荐阅读顺序”走。

## 材料说明

- `中国住房市场推演发布收口摘要_20260418_通俗版.md`
  - 给外行先看，尽量不用程序术语。
- `发布说明_20260418.md`
  - 给需要完整理解项目定位、核心机制和关键结果的人看。

  - 给正式演示前做流程对照的人看。
