# Attribution

This repository contains a real-estate simulation system that was initially
inspired by and evolved from the open-source `camel-ai/oasis` project:
https://github.com/camel-ai/oasis

Current positioning:

- This is not the official OASIS repository.
- This is not an official CAMEL-AI release.
- This repository has diverged from the upstream social-media simulation use
  case and now focuses on real-estate trading, finance, mortgage risk, market
  bulletin generation, and experiment orchestration.

Recommended external-facing description:

`This project was initially inspired by and evolved from CAMEL-AI's OASIS
project, and has since been expanded into an independent real-estate
simulation system.`

Branding guidance:
